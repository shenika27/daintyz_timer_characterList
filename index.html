🎨 캐릭터 타이머 스킨 빌더

캐릭터·타이머·미리보기·catalog 정보를 모아 번들 {skinId}.zip 하나로 만듭니다. 이 zip을 디자인레포 _inbox/에 업로드하면 GitHub Action이 자동으로 풀어 character_zip/·preview/ 배치 + catalog.json 병합까지 해줍니다. 코드/복붙 작업 없음.

1. 기본 정보

skinId는 영문/숫자만. 디자인레포의 character_zip/{skinId}.zip, catalog.json의 skinId와 반드시 일치해야 합니다.

→ 무료 테마

체크 시 '업데이트 평생이용권'으로도 해금되지 않고 항상 개별구매로만 풀립니다(상점에 별도 표시).

가격은 skin.json이 아니라 catalog.json에 들어갑니다. zip을 만들면 아래에 catalog에 붙여넣을 줄을 함께 만들어 드려요.

2. 캐릭터 상태별 프레임

상태별로 이미지를 올리면 순서대로 stop_01.png, stop_02.png… 처럼 자동 명명됩니다. 프레임 1장이면 정지 이미지, 여러 장이면 그 간격(ms)으로 애니메이션됩니다. 프레임은 투명 배경 PNG/WebP·한 변 1024px 이하만 받습니다.

3. 타이머 디자인

모든 테마는 타이머 디자인을 포함합니다(필수). 배경·버튼·숫자폰트를 지정하세요. 비워둔 항목은 앱 내장 기본값으로 폴백됩니다.

타이머 배경 PNG

박스·구분선이 모두 그려진 한 장. 가로 레이아웃(시간 좌 / 버튼 우) 기준. 없으면 내장 박스로 폴백.

버튼 그림 (선택 — 올리면 buttonStyle = skin)

누락된 버튼은 앱 내장 기호로 자동 폴백됩니다. play/pause는 시작·일시정지 슬롯에서 상태에 따라 바뀝니다.

숫자 폰트 (선택)

.ttf를 올리면 숫자를 그 폰트로 렌더링합니다(이때 크기는 영역에 맞춰지고 sizeSp는 무시). 색만 바꾸려면 폰트는 비우고 색만 지정하세요.

4. 미리보기 이미지

상점/타이머 탭에 쓰일 이미지입니다. 썸네일은 필수(상점 목록·타이머 탭 공용), 미리보기는 여러 장 올리면 상세화면 갤러리에 순서대로 표시됩니다. 올린 파일은 자동으로 PNG로 변환·명명됩니다(thumb.png, prev01.png…).

썸네일 (thumb.png)

상점 카드·타이머 탭에 뜨는 대표 1장. 정사각형 권장(아니면 경고), 한 변 1024px 이하.

미리보기 갤러리 (prev01, prev02…)

상세화면에서 넘겨 보는 이미지들. 없으면 갤러리 생략(썸네일만).

＋ 이미지 끌어다 놓기 / 클릭해서 선택 (여러 장 가능)

    5. catalog.json 항목 (자동 포함)

    아래 항목이 번들 안 catalog_entry.json으로 들어가고, Action이 catalog.json의 "skins" 배열에 자동 추가/갱신합니다. 손으로 복붙할 필요는 없어요(아래 복사 버튼은 확인·수동 보정용).
