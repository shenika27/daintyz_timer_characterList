🎨 캐릭터 타이머 스킨 빌더

아래 항목을 채우고 [zip 만들기] → [업로드] 두 단계면 끝입니다. 나머지(배치·등록)는 자동으로 처리돼요.

1. 기본 정보

skinId는 테마의 고유 ID입니다(영문·숫자·_, 한 번 정하면 바꾸지 마세요). 기존 ID로 만들면 그 테마를 덮어씁니다(수정).

→ 무료 테마

개별 구매로만 풀리는 특별 테마로 표시됩니다(이용권 일괄해금 대상에서 제외).

상점 목록에서 빠집니다(기프트코드·링크 전용 스킨용). 이미 보유한 사용자의 창고에는 정상 표시됩니다.

업로드 즉시 판매가 시작됩니다. 종료일을 지정하면 그날까지만 구매 가능하고, 지나면 상점에 '기간만료'로 잠겨 표시됩니다(이미 보유한 사용자는 계속 사용). 비우면 상시 판매.

사용자가 앱 설정 → 기프트코드에 이 코드를 입력하면 테마가 해금됩니다. 영문·숫자·-만, 대소문자·공백은 무시돼요. 코드는 안전하게 보관되며 원문은 다시 볼 수 없으니 따로 기록해두세요.

2. 캐릭터 상태별 프레임

한 장이면 정지 이미지, 여러 장이면 아래 간격(ms)으로 애니메이션됩니다(미리보기로 바로 확인). 프레임은 투명 배경 PNG/WebP, 한 변 1024px 이하만 받습니다.

3. 타이머 디자인

모든 테마는 타이머 디자인을 포함합니다(필수). 배경·버튼·숫자폰트를 지정하세요. 비워둔 항목은 앱 내장 기본값으로 폴백됩니다.

타이머 배경 PNG

박스·구분선이 모두 그려진 한 장. 가로 레이아웃(시간 좌 / 버튼 우) 기준. 없으면 내장 박스로 폴백.

버튼 그림 (선택)

올리지 않은 버튼은 앱 기본 기호로 표시됩니다. play/pause는 시작·일시정지 상태에 따라 자동으로 바뀝니다.

숫자 폰트 (선택)

.ttf를 올리면 숫자를 그 폰트로 렌더링합니다(이때 크기는 영역에 맞춰지고 sizeSp는 무시). 색만 바꾸려면 폰트는 비우고 색만 지정하세요.

4. 미리보기 이미지

썸네일은 필수입니다(상점 목록·타이머 탭에 뜨는 대표 이미지). 미리보기는 여러 장 올리면 상세화면 갤러리에 순서대로 표시됩니다.

썸네일 (thumb.png)

상점 카드·타이머 탭에 뜨는 대표 1장. 정사각형 권장(아니면 경고), 한 변 1024px 이하.

미리보기 갤러리 (prev01, prev02…)

상세화면에서 넘겨 보는 이미지들. 없으면 갤러리 생략(썸네일만).

＋ 이미지 끌어다 놓기 / 클릭해서 선택 (여러 장 가능)
